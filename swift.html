SWIFT Code 网站小知识

1、如何查找或查询 SWIFT Code 信息？

需要找到企业或个人的银行 SWIFT Code 进行收款时，可在搜索框输入国家、城市、银行名称、分行名称的全部或部分信息进行查找；若想核实已收到的 SWIFT Code 或查询对应地址，可直接输入该 SWIFT Code。若所查询城市没有单独的代码，可尝试省会城市或省会所在分行。

2、什么是 SWIFT Code？

SWIFT Code（也称 BIC）是银行识别码的标准格式，用于指定特定银行或分支机构，常用于国际电汇和银行间信息交换。代码由 8 或 11 个字符组成，格式为 AAAA BB CC DDD，其中 11 位或以“XXX”结尾的 11 位表示总行或主要办公室。

  • 银行代码（Bank Code）：4 位英文字母，通常是银行名称缩写。
  • 国家代码（Country Code）：2 位英文字母，标识所在国家或地区。
  • 地区代码（Location Code）：2 位数字或字母，区分地理位置，如省、州或城市。
  • 分行代码（Branch Code）：3 位字母或数字，用来区分分行；若无分行代码，默认“XXX”。

示例 BKCHCNBJ300 表示：BKCH（银行代码）、CN（国家代码）、BJ（地区代码）、300（分行代码）。SWIFT 代码由全球银行间金融电信协会（SWIFT）注册管理，总部位于比利时拉胡普。

3、SWIFT Code 作用是什么？

SWIFT Code 是国际转账时用于识别收款银行或分行的业务标识符，确保电汇及 SEPA 付款安全准确送达，属于全球公认的跨境汇款标准格式。

4、国内银行代码（Domestic Bank Codes）

部分国家在本国境内还实施国内银行代码或清算系统，例如：

  • 美国：Routing Number
  • 加拿大：Routing Number、Transit Number
  • 英国：Sort Codes
  • 爱尔兰：National Sort Codes (NSC)
  • 德国：BLZ Codes
  • 瑞士：BC 代码、SIC 代码
  • 法国：Banque & Code Guichet
  • 意大利：ABI 与 CAB Code
  • 澳大利亚/新西兰：BSB number
  • 印度：IFSC Code
